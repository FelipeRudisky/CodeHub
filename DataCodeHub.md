### Proyectos Disponibles:

**1.- Curriculum Vitae (CV).html**
Descripción: Mi Curriculum Vitae en formato web moderno. Implementado con HTML y CSS Grid puro, el proyecto incluye una funcionalidad clave: **la generación y descarga de un archivo PDF** optimizado directamente en el navegador (client-side), utilizando las librerías `html2canvas` y `jspdf`. Ideal para desarrolladores que buscan un ejemplo de despliegue de CV y conversión a PDF.
